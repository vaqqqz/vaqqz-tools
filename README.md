# 🖥️ Минималистичное и быстрое веб-приложение для сбора прокси-конфигураций из различных источников

![Status](https://img.shields.io/badge/Status-Active-success?style=flat-square)
![Platform](https://img.shields.io/badge/Platform-Web-white?style=flat-square)
![License](https://img.shields.io/badge/License-MIT-black?style=flat-square)

---

## ⚡️ Функционал
* Автоматическое удаление дубликатов и лишнего текста из списков

## 🛠️ Поддерживаемые протоколы

Программа фильтрует и собирает следующие типы ссылок:
- `VLESS://`
- `VMess://`
- `Trojan://`
- `Shadowsocks (SS)://`

---

## Запуск
* Зайдите на сайт https://vaqqqz.github.io
* Загрузите несколько ссылок на разные подписки
* Скопируйте список конфигураций, либо же скачайте файл с ними
